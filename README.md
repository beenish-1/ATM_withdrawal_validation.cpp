ATM Withdrawal Validation — C++

A simple banking logic program that validates ATM withdrawal requests using multiple conditions.

Features

Checks if the withdrawal amount is a multiple of 500
Ensures the amount does not exceed the daily limit of 20,000 PKR
Confirms the account balance is sufficient
Displays clear, meaningful error messages
Returns true for valid withdrawal, otherwise false
